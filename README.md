# Apple Frameworks

## Overview

This project was created as a SwiftUI refactoring challenge.

The original application displayed Apple frameworks in a `LazyVGrid`. The goal of the challenge was to refactor the UI and navigation flow by replacing the grid layout with a `List` while preserving the existing functionality and improving the overall code structure.

## Challenge Goals

- Refactor a grid-based interface into a list-based interface
- Migrate navigation to `NavigationStack`
- Implement value-based navigation using `NavigationLink(value:)`
- Improve code readability and maintainability
- Keep UI components reusable
- Preserve the existing user experience

## Features

- Browse Apple frameworks in a native SwiftUI `List`
- Navigate to a framework detail screen
- View framework descriptions
- Open Apple's official documentation inside the app
- Reusable SwiftUI components

## Technologies

- Swift
- SwiftUI
- NavigationStack
- NavigationLink
- SafariServices
- MVVM
